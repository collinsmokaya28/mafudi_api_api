Food Delivery Platform API Documentation

Restaurant Endpoints

GET /restaurants

Get a list of all restaurants.

Request:

No parameters required.

Response:

JSON array of restaurant objects.

Authentication:

Not required.

GET /restaurants/:id

Get details of a specific restaurant.

Request:

No parameters required.

Response:

JSON object representing the restaurant.

Authentication:

Not required.

POST /restaurants

Create a new restaurant.

Request:

JSON object representing the new restaurant.

Response:

JSON object representing the created restaurant.

Authentication:

Required (admin role).

PATCH /restaurants/:id

Update an existing restaurant.

Request:

JSON object representing the updated restaurant.

Response:

JSON object representing the updated restaurant.

Authentication:

Required (admin role).

DELETE /restaurants/:id

Delete a restaurant.

Request:

No parameters required.

Response:

No content.

Authentication:

Required (admin role).

Menu Item Endpoints

GET /menu_items

Get a list of all menu items.

Request:

No parameters required.

Response:

JSON array of menu item objects.

Authentication:

Not required.
